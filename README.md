# Rite of Spring

Rite of Spring is a personal project 

## Installation

Use the package manager [node](https://www.npmjs.com/get-npm) to install Jekyll.

```bash
npm i jekyll
```

## Usage

```bash
gem install jekyll bundler

bundle exec jekyll serve
```
